# Typed-text
Types html text from vanilla javascript and erase it in same fashion.
You can ommit the css. 
All you have to do is import the array and run it through the function.

Import the function typedText

Put your array in the first paramentr 
put the call or id of the wrapper in the secoend parameter
put the time interval in the third parameter.

If you don't give any time for the interval it will work with the default value of 300ms.
